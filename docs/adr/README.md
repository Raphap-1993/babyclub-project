# Architecture Decision Records

Este directorio contiene las decisiones arquitectónicas del proyecto.
Generadas por el agente `/orchestrate` y documentadas por `/docs`.

## Formato de nombre
`ADR-[NNN]-[descripcion-en-kebab-case].md`

## Estados posibles
- **Propuesto** — en revisión
- **Aprobado** — vigente
- **Rechazado** — evaluado y descartado
- **Deprecado** — reemplazado por otro ADR
- **Superseded por ADR-[N]** — reemplazado

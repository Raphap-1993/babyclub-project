# Decision Notes

Usa esta carpeta cuando una decision de trabajo necesite mas detalle que el log central.

Convencion recomendada:

- `DEC-XXXX-slug.md`

Regla:

- Si la decision cambia arquitectura, datos, seguridad, tenancy o contratos, debe evaluarse promocionarla a ADR.
- El log vivo sigue en [../decisions.md](../decisions.md).
